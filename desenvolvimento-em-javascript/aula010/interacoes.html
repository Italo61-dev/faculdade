Eventos em JavaScript

Eventos de Teclado

Para testar o funcionamento dos eventos de teclado, clique sobre um dos campos de texto e pressione qualquer tecla. O resultado será apresentado conforme as características de cada evento.

Evento (KeyUp) - Disparado sempre que o usuário solta a tecla pressionada.

Evento (KeyDown) - Disparado sempre que o usuário pressiona a tecla.

Evento (KeyPress) - Este evento não é compatível com algumas versões de navegador e somente é disparado por uma parte das teclas do teclado.

Nome do Evento Uso inline Uso com addEventListener
KeyUp onkeyup keyup
KeyDown onkeydown keydown
KeyPress onkeypress keypress
